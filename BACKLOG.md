# Backlog de mejoras

Ideas que quedan guardadas para futuras tandas.

## Proxima tanda recomendada

- Historial de precios por producto: ultimo precio, minimo, maximo y variacion.
- Comparador durante la compra: verde si esta barato, amarillo normal, rojo caro.
- Autocompletar lista con productos frecuentes y cantidades habituales.
- Filtros rapidos en stock: solo faltantes, por rubro y con foto/sin foto.

## Informes

- PDF con graficos simples por categoria.
- Ranking de productos mas comprados.
- Ranking de rubros con mayor gasto.
- Vista previa antes de compartir PDF.

## Backup familiar

- Vista previa al importar: compras, productos, stock y lista incluidos.
- Mostrar fecha del ultimo backup importado.
- Mensaje mas claro para enviar datos a otro celular.

## Experiencia visual

- Hacer tarjetas de producto mas compactas para compras largas.
